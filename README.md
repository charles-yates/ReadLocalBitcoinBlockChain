# ReadLocalBitcoinBlockChain
C# script to read a local bitcoin blockchain

# Different Implimentations
| Method | Speed to Setup | Storage Required | Best For | 
|---|---|---|---|
| Public API | Instant | 0 GB | Simple balance checks or lightweight wallet apps. |
| RPC Node Client | Medium  (Requires Node Sync) | ~600+ GB | Secure transaction signing, trading bots, and hosting services. |
| Direct Node Client | Fast | ~600+GB | Large-scale data science, historical research, and analytics. |
